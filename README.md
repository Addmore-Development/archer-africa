# Archer Africa Website

## Overview
A multi-page static website for Archer Africa, an international academic experiences company offering immersive learning programs across Africa. The site markets educational adventures to students and academic institutions.

## Tech Stack
- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Hosting:** Static file hosting (Render / similar)
- **Assets:** Custom SVGs (animal icons, footprint graphics), Google Fonts

## Pages
- `index.html` — Homepage / landing
- `about.html` — About Archer Africa
- `why-africa.html` — Value proposition for Africa-based programs

## Key Features
- Collapsible service cards with animal/footprint icons
- Numbered card layout with color-ordered sequencing
- "Work With Us" nav tab (merged from "How We Work" + "Contact")
- Cleaned hero sections across all pages
- Responsive multi-section layout
